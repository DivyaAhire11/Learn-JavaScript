TODAY LEARN ABOUT CONSOLE METHODS

1] console.log()  :- use of log the output in console
                   eg.  console.log("Hello World!");
                        console.log("value of x is",x);

2] console.info()  :- logs informational messages(same as console.log() in most browsers).
                   eg.  console.log("This is an info message");
                        
3] console.warn()  :- display warnings in the console,often with a yellow icon
                   eg.  console.log("this is a warnings");

4] console.error()  :- display error messages in red, often used to log actual errors
                   eg.  console.log("Something went wrong!");
                        console.log("this is an error");

5] console.table()  :- display tabular data (array or object) in a table format
                   eg.  const user = [{name:"rina",age=25},{name:"rina",age=25}]
                        console.table(user);

6] console.clear() :- clear the console 
                   eg. console.clear()
